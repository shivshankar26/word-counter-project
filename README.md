# 🚀 Word Counter Project

## 📌 Overview

This is a simple Python project that reads a text file and counts:

* Total number of words
* Frequency of each word

---

## 🛠️ Technologies Used

* Python 3.10
* Git & GitHub
* VS Code

---

├── screenshot/

├── README.md
└── requirements.txt

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/word-counter-project.git
```

2. Navigate to project:

```
cd word-counter-project
```

3. Run the program:

```
python src/main.py
```
